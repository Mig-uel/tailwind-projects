Arbitrary Colors:

bg-[#427fab]
Notes:
  • Black and white don't have shades
  • Shades for other colors range from 50-900
  • inputs must have class of border
  • buttons must have class of outline
  • Elements with a shadow must have default class of shadow-[size]
  • Opacity of shadowed elements can be set after the shadow color shadow-[color]-[shade-number]/[opacity]
  • When using a hex color in the arbitrary color class, you can add a 4th pair which will give you opacity: bg-[#33669966]
  • you can define variables for colors in a style tag (or external CSS file that you import) and you can use those variables: bg-[var(--mycolor)]
